Willkommen,

die Idee dieser Webseite und der dazugehörigen Ergebnisse sind im Rahmen eines Moduls an der Beuth Hochschule entstanden. Mittels einer Webkarte wird das Eintreten in die genannte virtuelle Welt möglich, hierzu wird ein entsprechendes Head Mounted Display(VR-Brille) benötigt. Den Benutzer erwarten zwei verschiedene Herangehensweisen. Bei der ersten Variante können durch das Auswählen eines vorgegebenen Standortes die erstellten 3D-Objekte betrachtet werden. Die Hintergrundvisualisierung ist hierbei durch eine 360° Bildaufnahme realisiert. Die andere Methode beinhaltet eine virtuelle Umgebung, in der mithilfe eines Controllers eine Fortbewegung möglich ist. Diese Umgebung kann durch einen Mausklick an einen beliebigen Ort auf der Karte erzeugt werden.
